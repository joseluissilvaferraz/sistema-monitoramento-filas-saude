# Sistema de Monitoramento de Filas em Unidades de Saúde

## Projeto Final - IFSP Campus Campinas

Autor: José Luis da Silva Ferraz

## Descrição

Este projeto foi desenvolvido em linguagem C utilizando a estrutura de dados Fila (FIFO - First In, First Out) para organizar o atendimento de pacientes em unidades de saúde.

## Funcionalidades

- Cadastro de pacientes
- Cadastro de salas
- Inclusão na fila
- Visualização da fila
- Painel de chamadas
- Chamada do próximo paciente

## Tecnologias

- Linguagem C
- Code::Blocks
- GCC
- Windows 11
- GitHub
